\doxysubsection{Detailed Description}
Represents equation(ax\texorpdfstring{$^\wedge$}{\string^}2 + bx + c = 0), it\textquotesingle{}s solution and type.~\newline
If equation has no roots, x1 and x2 contains 0.~\newline
If equation is linear, x1 contains root, x2 contains 0.~\newline
If equation is quadratic, x1 and x2 contain roots (x1 \texorpdfstring{$<$}{<} x2).
