CLI utility that can solve square equation in realtime user interaction.\hypertarget{md__r_e_a_d_m_e_autotoc_md0}{}\doxysection{\texorpdfstring{Installing}{Installing}}\label{md__r_e_a_d_m_e_autotoc_md0}
Supporting only UNIX-\/like systems.

Firstly, clone repo locally. For that open terminal and paste\+: \begin{DoxyVerb}git clone https://github.com/mikhio/SquareSolver
cd SquareSolver
\end{DoxyVerb}


Secondly, build with Makefile\+: \begin{DoxyVerb}make
\end{DoxyVerb}


And now in {\ttfamily ./build} folder will be builded utility. Can check installation like this\+: \begin{DoxyVerb}build/SquareSolver --help
\end{DoxyVerb}


If you see help info, all builded normally.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Get started}{Get started}}\label{md__r_e_a_d_m_e_autotoc_md1}
If utility builed normally, then you can use it!

Default mode is LOOP. It means you will see constantly {\ttfamily Enter a b c or q (to quit)\+:} and give endless inputs untill {\ttfamily q} or {\ttfamily quit} command or if max amount of error attempts will be reached.

Also can be in modes ONCE\+\_\+\+WITH\+\_\+\+ATTEMPTS and ONCE\+\_\+\+WITHOUT\+\_\+\+ATTEMPTS and run tests from {\ttfamily tests/ss\+\_\+tests.\+txt} check {\ttfamily -\/-\/help}.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Contributing}{Contributing}}\label{md__r_e_a_d_m_e_autotoc_md2}


It\textquotesingle{}s fully open source project and anyone can join to development!
