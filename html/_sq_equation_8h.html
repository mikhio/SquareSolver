◆ calcDiscriminant()

ReturnCode calcDiscriminant ( SqEquation * eq)

Calculates discrimminant.

Needs eq->type to be equal SQUARE
Writes D = b^2 - 4ac to eq->D
If D < 0, then changes eq->type to D_NEGATIVE
If D == 0, then changes eq->type to FULL_SQUARE
If D > 0, then eq->type remains SQUARE
