Detailed Description

Represents equation(ax^2 + bx + c = 0), it's solution and type.
If equation has no roots, x1 and x2 contains 0.
If equation is linear, x1 contains root, x2 contains 0.
If equation is quadratic, x1 and x2 contain roots (x1 < x2).
